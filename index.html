Cypress_Typescript

Technologie: Cypress, TypeScript, JavaScript

W ramach tego projektu stworzyłem zestaw automatycznych testów end-to-end dla przykładowego sklepu internetowego FakeStore, testując kluczowe ścieżki użytkownika w aplikacji. Testy zostały napisane zgodnie z dobrymi praktykami automatyzacji np. każdy test jest niezależny, asercje sprawdzające nie tylko obecność elementu, ale też treść lub stan, tworzenie własnych komend dla powtarzalnych elementów.

Olimpic Games Project

Technologie: C#, ASP.NET MVC, Entity Framework

Aplikacja webowa stworzona w architekturze MVC z podłączoną bazą danych z DataBaseStar (olympics.db). Umożliwia przeglądanie danych o sportowcach, konkurencjach oraz zdobytych medalach. Zalogowany użytkownik ma również możliwość dodania nowego udziału sportowca w wydarzeniu.
